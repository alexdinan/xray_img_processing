Running the program:
	python main.py path_to_source_images

Execution time:
	- ~ 7.5 mins for 100 images (256x256)
	- ~ 4.7 seconds per image (256x256)

Python Files:
	- inpainter.py implements custom image inpainting via Inpainter class
	- main.py imports Inpainter. Applies image processing to input images and saves results in ./Results directory
